Easily create a Product Directory.

== Description ==

If you’re looking to become a local product authority, the WordPress Product Directory plugin by CreativeMinds is the one for you. This plugin gives you the ability to create an embeddable directory for your WordPress site. The extra addition of an Add-On will support new entries and editable entries by your site users who wish to be listed.
